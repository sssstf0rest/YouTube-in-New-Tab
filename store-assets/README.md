# Chrome Web Store Asset Pack

This folder contains the current canonical asset pack for the Chrome Web Store listing.

These files are the current standard and should be treated as the authoritative submission assets for this project.

## Current Standard Files

- `../icons/icon-128.png`
  Chrome Web Store icon
- `design 440 x 280.png`
  Small promo tile
- `design 1440 x 560.png`
  Wide promo graphic file used as the marquee-style store graphic
- `design 1280 x 800.png`
  Store screenshot
- `design 3840 x 2160.png`
  High-resolution master artwork
- `design.pptx`
  Editable source deck

## Measured File Sizes

- `design 440 x 280.png`: `440x280`
- `design 1440 x 560.png`: actual image size `1400x560`
- `design 1280 x 800.png`: `1280x800`
- `design 3840 x 2160.png`: `3840x2160`

## Notes

- The old `screenshots/` subfolder is no longer used.
- If you update these design files manually, keep this README and the store submission guide in sync.
