Privacy Policy

YouTube in New Tab

This Privacy Policy explains what information the YouTube in New Tab Chrome extension accesses, stores, and uses.

1. Summary

YouTube in New Tab is a Chrome extension that changes how supported YouTube video links open. When enabled, a normal left-click on a supported YouTube /watch link opens that video in a new tab instead of replacing the current page.

This extension does not sell personal information, does not use analytics, does not use advertising trackers, and does not send your browsing data to the developer's servers.

2. Information the Extension Accesses

The extension runs only on https://www.youtube.com/*. On those pages, it checks click events so it can detect supported YouTube video links and open them in a new tab.

  • The URL of the YouTube video link you clicked
  • The current tab context needed to open the new tab beside it
  • Your extension preference stored as { enabled: boolean }

3. Information the Extension Stores

The extension stores only one setting: whether the feature is enabled or disabled.

  • Storage location: chrome.storage.sync
  • Stored value: { enabled: true | false }

If Chrome Sync is enabled in your browser profile, this setting may sync between your signed-in Chrome browsers through Google's Chrome sync infrastructure.

4. How Information Is Used

The extension uses the information above only to provide its feature:

  • Detecting supported YouTube video link clicks
  • Opening the clicked video in a new tab beside the current tab
  • Remembering whether the feature is turned on or off

5. What the Extension Does Not Do

  • It does not collect account credentials, passwords, or payment information.
  • It does not collect or transmit personal messages, emails, or form entries.
  • It does not sell or share personal data with third parties.
  • It does not run ads or use third-party analytics tools.
  • It does not send your YouTube activity or browsing history to the developer.

6. Third-Party Services

The extension does not use any developer-operated backend or third-party analytics service. The only external infrastructure involved is the standard Chrome extension platform and, if enabled by you, Chrome Sync.

7. Data Sharing and Sale

The developer does not sell, rent, or trade user data. The extension does not share user data with third parties for advertising, profiling, or marketing purposes.

8. Security

The extension is designed to minimize data handling. It stores only a small settings value and does not transmit user data to developer servers.

9. Changes to This Policy

This Privacy Policy may be updated if the extension's functionality or data practices change. Any updated version should be published at the same public URL used for the Chrome Web Store listing.
